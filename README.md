# Vue3 Navigation Bar

Collapsable, Responsive, Backdrop and with Transition.

## Info

Uses Vue3 Composition API to for the Navbar state.<br/>
Uses Font-Awesome Free icons.
<br/>

## Project setup

```
npm install
```
